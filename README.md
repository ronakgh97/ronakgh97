#### some of my works:
- [blazedb](https://github.com/ronakgh97/blaze-db): disk & memory optimized vector search engine/database, which uses [hnsw](https://arxiv.org/pdf/2512.06636) + [ivf](https://www.velodb.io/glossary/inverted-file-index-in-vector-similarity-search) + [[turboquant](https://arxiv.org/pdf/2504.19874) (WIP)] and simd for fast retrivals.
- [ghost-sync](https://github.com/ronakgh97/ghost-sync): composable async network rust library for [multiplayer integration](https://www.gabrielgambetta.com/client-server-game-architecture.html) in game dev.
- [rdrive](https://github.com/ronakgh97/rdrive): decentralized and secure zero-trust object storage node, which uses CAS/Layering with versioning & backups support, essentially amazon s3 and blackblaze b2 clone minus the "bloat".
- [hnsw](https://github.com/ronakgh97/blaze-db): simplified library implementation of hierarchical NSW from [paper](https://arxiv.org/pdf/1603.09320) in rust.
- [blas](https://github.com/ronakgh97/blas_rs): [openBlas](https://github.com/OpenMathLib/OpenBLAS) kernels implementation in rust for x86_64 arch only.

my interests: database & system design and patterns, micro-opt, hcp, crypto, compilers, game dev.......and pokemons
